Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select an element in DOM. Those ways are:

  1. getElementById.
  2. getElementsByClassName.
  3. getElementsByTagName.
  4. querySelector.
  5. querySelectorAll.

Question-3: What is event delegation in the DOM?

When we add event listener is a childNode, in the time of calling this event it will listen from all of his parents node. To prevent this we use event listener in the parentNode for avoiding adding multiple event listener in the childNode.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a event propagation methods. its like when we trigger an event from a child node same event will be triggered on its parentNode than if parentNode have any parentNode i will trigger that node also. to stop event propagation we use event delegation methods.
